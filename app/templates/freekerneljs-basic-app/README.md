It's a basic application in the following structure:

```
.
└── app
    ├── index.html
    ├── app.bootstrap.js
    ├── app.module.js
    ├── app.routes.js
    ├── assets
    │   ├── css
    │   └── images
    ├── widgets
    │   ├── header
    │   │   ├── module.js
    │   │   └── template.html
    │   └── navbar
    │       ├── module.js
    │       └── template.html
    ├── views
    │   └── home

```

- angular (v1.3.13)

- angular-ui-router (v0.2.13)
